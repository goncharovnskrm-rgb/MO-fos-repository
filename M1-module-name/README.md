# Модуль 1. Методы обучения с учителем


Директория содержит MD-файлы с описаниями КИМ для модуля 1. В этом файле можно сформировать таблицу со списком КИМ для модуля, их краткими аннотациями и ссылками на соответствующие MD-файлы.


# Модуль 1. КИМ (Контрольно-измерительные материалы)

| № | Код КИМ | Дескриптор (формулировка) | Ссылка |
|---|---|---|---|
| 1 | ML-2.3-Б | Способен сделать адекватный выбор основных метрик оценки качества для задач классификации и регрессии в зависимости от баланса классов, наличия выбросов в данных и бизнес-требований. | [ML-2.3-Б](ML-2.3-Б.md) |
| 2 | ML-2.3-С | Способен сделать адекватный выбор типов кросс-валидации в зависимости от объёма данных, баланса классов и наличия временной структуры в датасете чтобы избежать переобучения. | [ML-2.3-С-1](ML-2.3-С-1.md) |
| 3 | ML-2.3-С | Способен адекватно оценить качество ML-модели с учетом компромисса между смещением и дисперсией, а также способность модели к обобщению. | [ML-2.3-С-2](ML-2.3-С-2.md) |
