**3\. Данные**

В лабораторной работе №1 используются два датасета, которые загружаются автоматически через API scikit-learn:

**3.1. Bike Sharing Dataset (задача регрессии)**

| **Параметр** | **Значение** |
| --- | --- |
| **Источник** | UCI Machine Learning Repository, загружается через sklearn.datasets.fetch_openml(data_id=42712) |
| **Объём** | ~17 379 объектов, 12 признаков |
| **Целевая переменная** | cnt – общее количество прокатов велосипедов (непрерывная, регрессия) |
| **Типы признаков** | Смешанные: числовые (temp, feel_temp, humidity, windspeed, hour, month, weekday, year) и категориальные (season, holiday, workingday, weather) |
| **Особенности** | Наличие нелинейных зависимостей (влияние температуры и часа дня на спрос), сезонные паттерны. Требует One-Hot Encoding для категориальных признаков и масштабирования числовых. |
| **Размер на диске** | ~2–5 МБ (кэшируется в ~/scikit_learn_data/openml/) |

**3.2. Covertype Dataset (задача классификации)**

| **Параметр** | **Значение** |
| --- | --- |
| **Источник** | UCI Machine Learning Repository, загружается через sklearn.datasets.fetch_covtype() |
| **Объём (полный)** | 581 012 объектов, 54 признака |
| **Объём (в работе)** | 100 000 объектов (случайная подвыборка, random_state=42) |
| **Целевая переменная** | Cover_Type – тип лесного покрова, 7 классов (целочисленная, 1–7) |
| **Типы признаков** | Все числовые: 10 количественных (Elevation, Aspect, Slope, расстояния до гидрологии/дорог/пожаров, Hillshade) + 4 бинарных (Wilderness_Area) + 40 бинарных (Soil_Type) |
| **Особенности** | Сильный дисбаланс классов (класс 2 – ~283 тыс., класс 4 – ~2.7 тыс. в полном датасете). Именно поэтому в работе используется Weighted F1-score, а не Accuracy как основная метрика. Требует масштабирования. |
| **Размер на диске** | ~70–80 МБ (кэшируется в ~/scikit_learn_data/covertype/) |

**Примечание для преподавателя:**

- Первая загрузка Covertype может занять 1–3 минуты в зависимости от скорости интернета. Рекомендуется попросить студентов запустить ячейку загрузки заранее (в начале занятия).
- Если интернет-соединение в аудитории нестабильно, преподаватель может заранее загрузить оба датасета на одном компьютере и сохранить их в формате .csv или .parquet, а затем раздать студентам. В этом случае студентам потребуется заменить fetch_openml / fetch_covtype на pd.read_csv(...).
- Подвыборка 100000 объектов из Covertype фиксирована (np.random.seed(42)), поэтому результаты у всех студентов должны быть **идентичны** при одинаковых параметрах моделей. Это упрощает проверку.
